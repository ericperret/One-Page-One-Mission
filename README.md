# One-Page-One-Mission – Drone Audio Tracker
Ce projet est une interface web légère, autonome, qui suit en temps réel un drone par son bruit moteur. 
Rien à installer, rien à configurer : juste ouvre la page, et le tracking démarre. ## Comment ça marche ? 
Tout est fait en HTML et JavaScript pur – ça tourne directement dans ton navigateur. 
Pas de serveur, pas d'API, pas de téléchargement lourd. 
Juste toi, ton ordi et un micro. ## Tester ça, c'est simple : 
1. Clique sur un fichier HTML (genre v3.4).
2. Télécharge-le ou ouvre-le en raw.
3. Ouvre avec Chrome ou Firefox.
4. Autorise l'accès au micro quand il demande.
5. Et hop – position, vitesse, bruit, tout s'affiche.
  
 ## Annexes techniques - `KHEOPS_v2.5.pdf` à `v3.4.pdf` : les maths derrière – filtre de Kalman, Doppler, correction du vent, ISO 9613, gestion SNR… -
 
 Même chose en `.docx` si t'as besoin de modifier. Tous les détails pour comprendre ou bidouiller plus loin.
